# Team Research Visualization

## Overview

This project analyzes historical Bitcoin price data from 2019 to 2022 to explore the relationship between opening and closing prices. The research focuses on data cleaning, statistical analysis, and visualization techniques to derive insights from cryptocurrency market data.

## My Contribution

As a team member, I contributed to the data analysis and visualization components of this project. I assisted with data preprocessing steps including handling null values using interpolation techniques, performed descriptive statistical analysis on the dataset, and helped create visualizations including histograms and scatter plots to explore price relationships.

## Methodology

The research followed these key steps:

- Data loading and preprocessing using R
- Null value handling using the zoo package for interpolation
- Descriptive statistical analysis of opening and closing prices
- Distribution analysis using histogram visualization
- Exponential decay curve fitting to understand price distribution
- Scatter plot analysis to examine relationship between Open and Close prices
- Spearman correlation testing to measure monotonic relationship

## Results

Key findings from the analysis:

- The dataset covers 3 years of historical Bitcoin price data from 2019 to 2022
- The relationship between Open and Close prices was analyzed using Spearman correlation
- Histogram visualization revealed the distribution of closing prices
- Exponential decay curve was fitted to understand price spread patterns

## Tech Stack

- R
- ggplot2 (base R graphics)
- zoo package for time series interpolation

## Files

- `teamResearch.R` - Main R script containing all data loading, preprocessing, analysis, and visualization code
- `Bitcoin CZ.csv` - Dataset containing historical Bitcoin price data (Open, Close, Volume, Date fields)
